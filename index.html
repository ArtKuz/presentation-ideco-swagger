Методы REST API

  • Create - POST
  • Read - GET
  • Update - PUT / PATCH

PUT vs. PATCH

  • PUT - полностью пересоздаёт ресурс по указанному id.
  • PATCH - частично обновляет состояние ресурса по указанному id.

Методы REST API

  • Delete - DELETE
